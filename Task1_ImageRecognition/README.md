# Task 1 - Image Recognition

This task uses Google Teachable Machine to classify images into:
- Phone
- Pen

Files:
- predict.py
- keras_model.h5
- labels.txt
- prediction_result.png

## Steps

1. Collected images for two classes (Phone and Pen).
2. Trained the model using Google Teachable Machine.
3. Exported the model in TensorFlow Keras format.
4. Wrote a Python script to load the model and classify an input image.
5. Tested the model using a phone image.
6. Saved the prediction result screenshot.
